# Frying-Pan-Loader-Animation-

A clean and minimal frying pan loader animation built using pure HTML and CSS. This project demonstrates how to create smooth, lightweight UI animations without using JavaScript.




📦 Technologies

Built with:

HTML: Markup language for creating the structure.
CSS: For styling and animations.




🛣️ Project Structure

Here are the key files and their purposes:

index.html: The main HTML file containing the structure of the Frying Pan Loader.
style.css: The CSS file that styles the pan and adds animations.
Feel free to explore and customize these files to suit your needs!




◼️ How It Works

The animation is created using CSS transforms and keyframes:

The pan rotates using the rotate() transform.

The food element moves vertically using the translateY() transform.

The animation runs infinitely using CSS animation properties.

The layout is centered using Flexbox.




💬 Concepts Used

CSS Flexbox for centering layout

CSS Positioning (relative and absolute)

CSS Animations (@keyframes)

CSS Transforms (rotate, translateY)

CSS Pseudo-elements (::after)



  
🏔️ Purpose

This project was created to practice CSS animations, layout positioning, and building custom loaders for modern web interfaces.

More improvements and variations will be added later as the project evolves.
